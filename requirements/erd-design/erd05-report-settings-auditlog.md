# Report Module

### reports

```
| Column       | Type         | Constraint    | Description    |
| ------------ | ------------ | ------------- | -------------- |
| id           | UUID         | PK            | Report ID      |
| generated_by | UUID         | FK → users.id | Generated by   |
| report_type  | ENUM         | NOT NULL      | Report type    |
| report_name  | VARCHAR(255) | NOT NULL      | Report name    |
| file_format  | ENUM         | NOT NULL      | Output format  |
| file_url     | TEXT         | NOT NULL      | Storage URL    |
| filters      | JSONB        | NULL          | Report filters |
| generated_at | TIMESTAMP    | DEFAULT now() | Generated time |
| created_at   | TIMESTAMP    | DEFAULT now() | Created time   |

```

Report Type

- EMPLOYEE
- ATTENDANCE
- LEAVE
- OVERTIME
- PAYROLL
- REIMBURSEMENT
- DOCUMENT

File Format

- PDF
- EXCEL
- CSV

| ----------- | ------------ | ------------- | ------------- |
| id          | UUID         | PK            | Setting ID    |

| updated_by  | UUID         | FK → users.id |               |
